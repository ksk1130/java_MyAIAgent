ChatCLI - ポータブル版

========================================
起動方法
========================================
run.bat をダブルクリックするか、以下をコマンドプロンプトで実行してください：
  run.bat

========================================
ファイル構成
========================================
run.bat      - アプリケーション起動スクリプト
lib/         - JAR ファイルとライブラリ
jre/         - Java 21 ポータブル実行環境

========================================
要件
========================================
- Windows 7 以上
- JRE は同梱済み（別途インストール不要）

========================================
トラブルシューティング
========================================
- コマンドプロンプトで run.bat を実行すると詳細なエラーメッセージが表示されます
